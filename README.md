# zskribidi

Port of [skribidi](https://github.com/memononen/Skribidi) to Zig, building all dependencies with zig. The bindings are very new so please report any issues you find.

## Building
Requires Zig 0.15.1 or later.

```sh
zig build
```

## Running tests
```sh
zig build test
```
